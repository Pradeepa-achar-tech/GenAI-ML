              explain: '**Deep Q-Network (DQN)** is Q-Learning with a neural network replacing the Q-table. Instead of storing one number per (state, action) pair, a neural net `Q_θ(s)` outputs Q values for all actions given the state. This lets RL scale to **continuous, high-dimensional states** like raw pixels — DQN famously learned 49 Atari games from screen pixels alone (DeepMind, 2015) using the same network architecture for every game. Two engineering tricks make it work: **experience replay** (sample uniformly from a buffer of past transitions) and a **target network** (a frozen copy of Q used in the bootstrap target).',
              analogy: 'Think of tabular Q-learning as keeping a **paper logbook** with one row per (state, action) pair. For Tic-Tac-Toe (~5,000 states) the logbook fits in your bag. For chess (10⁴⁰ states) or Atari Pong (256³ × 84² screen pixels), you would need more paper than exists in the universe.\n\n**DQN replaces the logbook with a sketch artist.** You show the artist a board (the state, e.g., 84x84 pixels), and the artist sketches a row of action-scores from memory of similar boards seen before. The artist is a neural network — millions of parameters, but a fixed size. New states they have never seen exactly still get reasonable scores because the artist generalises across visually similar states.\n\nBut training the artist is fragile. Two specific traps:\n\n**Trap 1: Sequential samples are correlated.** If you train on consecutive Pong frames, all 60 are nearly identical — the network overfits to the current scene and forgets the rest. Solution: a **replay buffer**. Store the last 1 million transitions in memory; at each training step, sample 32 random transitions from anywhere in the buffer.\n\n**Trap 2: The target keeps moving.** In tabular Q-learning, when you update `Q[s, a] ← target`, the target depends on `max Q[s_next, ...]` from the OTHER cells. With a network, updating `Q_θ` to fit the target also changes the target itself — chasing your own tail. Solution: a **target network** `Q_θ⁻`, a frozen copy of `Q_θ` updated only every 10,000 steps. Now the target stays still long enough for the main network to learn.\n\nThese two tricks are why DQN works and naive deep Q-learning does not. import torch.nn as nn
import torch.nn.functional as F
import numpy as np
import gymnasium as gym
from collections import deque
import random

device = torch.device("cuda" if torch.cuda.is_available() else "cpu")

# === Q-network ===
class QNet(nn.Module):
    def __init__(self, obs_dim, n_actions, hidden=128):
        super().__init__()
        self.net = nn.Sequential(
            nn.Linear(obs_dim, hidden), nn.ReLU(),
            nn.Linear(hidden, hidden),  nn.ReLU(),
            nn.Linear(hidden, n_actions),
        )
    def forward(self, x):
        return self.net(x)

# === Replay buffer ===
class ReplayBuffer:
    def __init__(self, capacity=100_000):
        self.buf = deque(maxlen=capacity)
    def add(self, s, a, r, s_next, done):
        self.buf.append((s, a, r, s_next, done))
    def sample(self, batch_size):
        batch = random.sample(self.buf, batch_size)
        s, a, r, s_next, done = zip(*batch)
        return (
            torch.tensor(np.array(s), dtype=torch.float32, device=device),
            torch.tensor(a, dtype=torch.long, device=device),
            torch.tensor(r, dtype=torch.float32, device=device),
            torch.tensor(np.array(s_next), dtype=torch.float32, device=device),
            torch.tensor(done, dtype=torch.float32, device=device),
        )
    def __len__(self): return len(self.buf)

# === Hyperparameters ===
env = gym.make("CartPole-v1")
obs_dim = env.observation_space.shape[0]   # 4
n_actions = env.action_space.n             # 2
gamma = 0.99
batch_size = 64
lr = 1e-3
buffer_size = 50_000
target_update_every = 500
total_steps = 30_000
eps_start, eps_end = 1.0, 0.05
decay_steps = 10_000
warmup = 1000

# === Networks + optimizer ===
q_main = QNet(obs_dim, n_actions).to(device)
q_target = QNet(obs_dim, n_actions).to(device)
q_target.load_state_dict(q_main.state_dict())
for p in q_target.parameters(): p.requires_grad = False

opt = torch.optim.Adam(q_main.parameters(), lr=lr)
buffer = ReplayBuffer(buffer_size)

# === Training loop ===
s, _ = env.reset(seed=0)
ep_return, returns_log = 0.0, []
for step in range(total_steps):
    eps = max(eps_end, eps_start - (eps_start - eps_end) * step / decay_steps)
    if random.random() < eps:
        a = env.action_space.sample()
    else:
        with torch.no_grad():
            a = int(q_main(torch.tensor(s, dtype=torch.float32, device=device)).argmax())

    s_next, r, term, trunc, _ = env.step(a)
    done = term or trunc

    s = s_next; ep_return += r

    if done:
        returns_log.append(ep_return)
        s, _ = env.reset()
        ep_return = 0.0

    # Train one step
    if len(buffer) > warmup:
        bs, ba, br, bs_next, bdone = buffer.sample(batch_size)
        with torch.no_grad():

            y = br + gamma * target_q * (1.0 - bdone)
        pred_q = q_main(bs).gather(1, ba.unsqueeze(1)).squeeze(1)
        loss = F.smooth_l1_loss(pred_q, y)         # Huber loss
        opt.zero_grad(); loss.backward()
        nn.utils.clip_grad_norm_(q_main.parameters(), 10.0)
        opt.step()

                '**Skipping the target network.** Bootstrapping `Q_main(s, a)` toward `max Q_main(s_next, ...)` makes targets chase the prediction — divergence in 1000 steps. Always use a frozen target network.',
                '**Tiny replay buffer.** A 1000-sample buffer is dominated by recent correlated transitions. CartPole tolerates 10k; Atari needs 1M. If training is unstable, the first thing to try is increasing the buffer.',
                '**Updating before warmup.** Starting gradient updates at step 1 trains on 1-2 transitions of garbage. Wait until the buffer has at least 1000 samples (CartPole) or 50k (Atari).',
                '**Wrong terminal masking.** `target = r + gamma * Q_target(s_next).max() * (1 - done)`. If you forget the `(1 - done)` you bootstrap value into terminal states. Note: mask only on **terminated**, not **truncated** — Gymnasium splits these for a reason.',
